## NeoVim

### ~/.config/nvim/init.vim

```
set shell=/bin/zsh
set shiftwidth=4
set tabstop=4
set expandtab
set textwidth=0
set autoindent
set hlsearch
set clipboard=unnamed
syntax on

call plug#begin()
Plug 'tpope/vim-commentary'
call plug#end()
```
